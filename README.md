# Face Recognition Attendance System

## Description
This project detects and recognizes faces using a webcam and marks attendance automatically.

## Features
- Real-time face detection
- Face recognition
- Attendance stored in CSV
- Multiple users supported

## Technologies
- Python
- OpenCV
- face_recognition
- NumPy

## How to Run
1. Add images in dataset folder
2. Run face_code.ipynb
3. Start camera
4. Attendance will be recorded

## Output
- Face detected with name
- Attendance saved in attendance.csv
